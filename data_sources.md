# Data sources

This project uses public processed or derived resources only. Large raw or processed matrices are not included in this GitHub package.

| Resource | Role in project | Public access |
| --- | --- | --- |
| GSE115978 | Primary melanoma single-cell Geneformer modeling and expanded in silico deletion input | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE115978 |
| GSE72056 | Processed-expression single-cell sensitivity resource only | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE72056 |
| GSE120575 | Immune-response-related melanoma single-cell resource; not used for malignant discovery | https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE120575 |
| TCGA-SKCM / GDC | Bulk expression and exploratory clinical association context | https://portal.gdc.cancer.gov/projects/TCGA-SKCM |
| cBioPortal | TCGA-SKCM access/context resource | https://www.cbioportal.org/ |
| DepMap / CCLE | Melanoma-lineage and pan-cancer dependency context | https://depmap.org/portal/download/ |
| GDSC2 | Exploratory cell-line drug-sensitivity association context | https://www.cancerrxgene.org/ |
| ChEMBL | Drug-target and bioactivity context | https://www.ebi.ac.uk/chembl/ |
| Open Targets | Target-disease association, tractability and known-drug context | https://platform.opentargets.org/ |
| DrugBank | Not completed as an automated evidence layer | DrugBank was not queried automatically and requires manual confirmation. |

Access dates, exact release versions and file names should be confirmed before final archival release.
